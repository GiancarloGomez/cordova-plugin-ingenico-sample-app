INGENICO SAMPLE APPLICATION SETUP REQUIREMENTS

It appears you do not have your credentials set properly.

  1. Create the following file JavaScript file
    src/assets/javascript/config/environment.js
  2. Enter your Ingenico API Credentials as displayed below and rebuild the application.
appSettings.ingenico.username  = "my-username";
appSettings.ingenico.password  = "my-password";
appSettings.ingenico.apiKey    = "my-api-key";
